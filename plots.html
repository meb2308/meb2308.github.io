Instacart

The following plots were generated using a random sample of the 2017 Instacart Online Grocery Orders dataset.

Setup and data download

data("instacart")

instacart = 
  instacart %>% 
  select(order_id, order_hour_of_day, reordered, user_id, product_id, product_name, department) %>% 
  sample_n(10000)

The sample contains 10000 randomly selected rows and 7 columns, with each row representing a single product from an instacart order. Variables include identifiers for the order and product, the day and time of the order, the grocery department, and the product name. In total, there are 4746 products found in 9467 orders.

Top selling departments

This plot shows the number of items ordered by department.

instacart %>% 
  count(department) %>% 
  mutate(
    department = fct_reorder(department, n)) %>%
  plot_ly(
    x = ~department, y = ~n, color = ~department,
    type = "bar", colors = "viridis")

Produce, dairy/eggs, and snacks are the top 3 departments from which orders come.

Boxplot

This plot shows the time of day (hour 0-24) when orderes were placed by department.

instacart %>% 
   mutate(
    department = fct_reorder(department, order_hour_of_day)) %>%
  plot_ly(
    x = ~department, y = ~order_hour_of_day, color = ~department,
    type = "box", colors = "viridis")

Most orders are placed in the late morning and the afternoon. Most departments see orders throughout the day, with the exception of a few departments, such as alcohol (where items are rarely ordered before 9am).

Proprtion reordered

This scatterplot shows the proportion of products that are re-ordered.

instacart %>% 
  group_by(product_name) %>% 
  summarize(proportion_reordered = mean(reordered), n=n()) %>% 
  plot_ly(
    x = ~n, y = ~proportion_reordered,
    alpha = .5, type = "scatter", mode = "markers")
## `summarise()` ungrouping output (override with `.groups` argument)

The plot indicates that, as we would expect, products that are ordered most often are also re-ordered most often, but there is a tapering effect.
